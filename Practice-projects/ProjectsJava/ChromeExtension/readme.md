### Leads Tracker Chrome Extension

A small Chrome extension I built while learning JavaScript.
It helps save useful links/leads either manually or directly from the current browser tab.
This project helped me practice:
-DOM manipulation
-Event listeners
-Arrays
-localStorage
-Chrome Extension APIs
-JSON methods
# How to Run
1-Download or clone the project
2-Open Chrome and go to chrome://extensions
3-Turn on Developer Mode
4-Click Load Unpacked
5-Select the project folder

The extension should now appear in Chrome like other extenssion.
